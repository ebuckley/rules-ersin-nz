What is this?

rules.ersin.nz is a knowledge base, and demo application built on the openfisca-aotearoa platform.

services

This demo application uses a few different services to provide example implementation on top of the Open Fisca platform.

Open Fisca Web Service

api.rules.ersin.nz is an instance of openfisca-aotearoa, extended with the openfisca-canterbury module. It provides a the default api access using the "openfisca serve" command.

Rating tool

rates.rules.ersin.nz is an example web application for showing how rules around rates calculation can be implemented using Open Fisca. The demo implements geospatial filtering of what variables need to be calculated for a specific area.

Legislation explorer

explorer.rules.ersin.nz is the legislation explorer project see `/src/legislation-explorer`. This service provides search through what rules have been implemented. It is an index of rules that can be used with rates.rules.ersin.nz

How it works.

This site is deployed on a single VPS of the smallest possible size. We use openfisca-aotearoa, python, flask, nodejs, express and plain old html to run the site.

This knowledge base is developed as an open source project, and you can read the code on github. Issues, Pull Requests and contributions are welcome.
